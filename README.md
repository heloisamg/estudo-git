# Estudo git

1.Configurar nome do usuário globalmente
```
git config --global user.name "Heloísa"
```

2.Configurar e-mail do usuário globalmente
```
git config --global user.email "teste@email.com"
```

3.Inicializar um repositório Git
```
git init
```

4.Verificar o status do repositório
```
git status
```

5.Adicionar um arquivo específico ao staging
```
git add README.md
```

6.Adicionar todos os arquivos modificados
```
git add .
```

7.Criar um commit com mensagem
```
git commit -m "Mensagem do commit"
```

8.Visualizar o histórico de commits
```
git log
```

9.Enviar alterações para o repositório remoto
```
git push
```

10.Atualizar o repositório local
```
git pull
```

11.Navegar entre diretórios
```
cd
```

12.Criar diretório
```
mkdir nome_da_pasta
```

13.Remover diretório vazio
```
rmdir nome_da_pasta
```

14.Remover arquivo
```
rm arquivo.txt
```

15.Exibir conteúdo de um arquivo
```
cat README.md
```

16.Editar arquivo no terminal
```
vi README.md
```

17.Clonar um repositório remoto para o computador local
```
git clone url_do_repositório
```
